# 신경망 스타일 전이 (Neural Style Transfer)

## 1. Gatys et al. (2015) — 최적화 기반

| 항목 | 내용 |
|---|---|
| 핵심 아이디어 | 콘텐츠 손실 + 스타일 손실을 동시에 최소화하여 이미지 픽셀을 직접 최적화 |
| 콘텐츠 특징 | VGG 깊은 층(conv4_2): 높은 수준의 구조·형태 정보 |
| 스타일 특징 | 여러 층의 그람 행렬(Gram Matrix): G = FᵀF / 채널 간 상관관계 |
| 그람 행렬 | G_ij = Σ F_ik · F_jk → 텍스처·색상 패턴 인코딩 |
| 총 손실 | L_total = α·L_content + β·L_style (α≪β 일반적) |
| 최적화 대상 | 출력 이미지 픽셀(가중치는 고정, 이미지를 역전파로 업데이트) |
| 단점 | 이미지 1장 생성에 수백 iteration 필요 → 느림(수 분) |

## 2. Fast Neural Style Transfer (Johnson et al.)

| 항목 | 내용 |
|---|---|
| 아이디어 | 스타일별 변환 네트워크를 사전 학습 → 추론 시 1회 순전파만 필요 |
| 구조 | Encoder → Residual Blocks → Decoder(Upsampling) |
| 속도 | 실시간 처리 가능(~1000× 빠름) |
| 단점 | 네트워크 1개 = 스타일 1개로 고정 |

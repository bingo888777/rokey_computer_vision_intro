# 3D 렌더링 (3D Vision & NeRF)

## 1. 3D 컴퓨터 비전 기초

| 개념 | 설명 | 활용 |
|---|---|---|
| 깊이 추정(Depth) | 단안/스테레오 카메라로 픽셀별 깊이 추정 | 자율주행, AR |
| 포인트 클라우드 | 3D 공간상의 점 집합(x,y,z[,r,g,b]) | LiDAR 처리 |
| 메시(Mesh) | 정점+면으로 구성된 3D 표면 표현 | 그래픽스, 게임 |
| 복셀(Voxel) | 3D 공간을 격자로 분할한 3D 픽셀 | 의료 영상, 충돌 |
| 카메라 캘리브레이션 | 내재(K), 외재(R,t) 행렬 추정 | 3D 재구성 전처리 |

## 2. NeRF (Neural Radiance Field)

| 항목 | 내용 |
|---|---|
| 핵심 아이디어 | 3D 장면을 MLP로 암묵적 표현: (x,y,z,θ,φ) → (RGB, σ) |
| 입력 | 3D 위치(x,y,z) + 시선 방향(θ,φ) |
| 출력 | 해당 위치의 색상(r,g,b) + 밀도 σ(불투명도) |
| 렌더링 | 체적 렌더링(Volume Rendering): 광선을 따라 적분 |
| 학습 | 다시점(Multi-view) 이미지 + 카메라 포즈 → 2D 재구성 손실 |
| 장점 | 임의 시점 합성, 매우 정밀한 3D 장면 표현 |
| 단점 | 학습·추론 매우 느림 → Instant-NGP, 3D Gaussian Splatting으로 개선 |

> 🔑 Positional Encoding: 저주파 좌표 입력을 고주파 푸리에 특징으로 변환 → 고주파 세부 묘사 학습 가능.

# AI 학습 정리 저장소

## 📌 소개
이 저장소는 **ROKEY 교육과정**에서 학습한 AI/딥러닝 관련 내용을 정리한 공간입니다.
강의에서 배운 이론을 스스로 재정리하고, 실습 코드를 통해 개념을 체화하는 과정을 기록합니다.

- 학습 출처: ROKEY (AI 교육과정)
- 목적: 학습 내용 정리 및 포트폴리오 자료화
- 상태: 현재 진행 중 (지속적으로 업데이트 예정)

## 📂 구성
주제(코드 기준)별로 폴더를 나누어 정리합니다.
├── 01_regression/              # 회귀 머신러닝
├── 02_classification/          # 분류 머신러닝
├── 03_cnn/                     # CNN
├── 04_pretrained_model/        # 사전학습 모델
├── 05_overfitting_prevention/  # 과적합 방지
├── 06_data_preprocessing/      # 데이터 전처리
├── 07_object_tracking/         # 추적 알고리즘
├── 08_style_transfer/          # 신경망 스타일 전이
├── 09_gan/                     # GAN
├── 10_lstm/                    # LSTM
├── 11_3d_rendering/            # 3D 렌더링
├── 12_ssd/                     # SSD (Object Detection)
│
├── each folder contains:
│   ├── summary.md      # 이론 요약 정리
│   └── example.ipynb   # 코드 예제

## 📝 요약 정리 안내
각 폴더의 `summary.md`는 ROKEY 강의 내용을 기반으로, 관련 자료를 함께 참고하여
제 언어로 재구성한 정리입니다. 단순 필기가 아닌 이해 위주로 작성하고 있습니다.
